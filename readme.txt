A complete Point of Sale and management system specifically designed for jewelry businesses.

== Description ==

JWPM Jewelry POS Manager is a comprehensive WordPress plugin that transforms your website into a powerful jewelry store management system. It handles everything from sales and inventory to customer management and installment tracking.

Key features:
*   **Point of Sale (POS):** A fast and intuitive interface to process sales.
*   **Inventory Management:** Keep track of your jewelry items, including weight, price, and SKU.
*   **Customer Management:** Maintain a detailed database of your customers.
*   **Installment Plans:** Easily manage and track customer installments.
*   **Repair Jobs:** Manage jewelry repair requests and their status.
*   **Custom Orders:** Handle special orders from clients.
*   **Reporting:** Get insights into your sales and performance.
*   **Multilingual Ready:** Easily translate the plugin into any language.

Perfect for small to medium-sized jewelry shops looking for an integrated solution within WordPress.

== Installation ==

1.  Upload the `jwpm-jewelry-pos-manager` folder to the `/wp-content/plugins/` directory on your server.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  A new menu 'Jewelry POS' will appear in your WordPress admin dashboard. Click on it to start managing your store.

== Frequently Asked Questions ==

= Can I use this plugin without any coding knowledge? =

Yes, the plugin is designed to be user-friendly and can be managed entirely from the admin dashboard.

= Does this plugin support barcode scanners? =

Support for barcode scanners is planned for a future version.

= How do I translate this plugin into my language? =

The plugin comes with a `.pot` file in the `/languages/` directory. You can use a tool like Poedit to create a translation for your language.
